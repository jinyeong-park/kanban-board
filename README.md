# 🚀 CoreStep: Free Kanban To-Do List & Job Tracker | JYNLAB

> **Daily Tasks, Career Milestones. One Board.**

**CoreStep** is a lightweight, web-based productivity tool designed for those who want to manage their daily grind and career growth in one place. It brings the simplicity of **Trello** and the career-tracking power of **Huntr** directly to your browser.

## 🎯 Why CoreStep?

Why switch between multiple apps when you can see everything at once? CoreStep is a "simple-is-best" solution for:

- **Today's Productivity:** Managing your daily to-do lists and routine tasks.
- **Future Career:** Tracking job applications, interview stages, and career milestones.

## ✨ Key Features

- **Zero-Installation:** Fully deployed on GitHub Pages. Use it anywhere, anytime through your browser.
- **Dual-Track Design:** Optimized columns for both general tasks and job status tracking.
- **Minimalist Interface:** A clean, distraction-free environment to keep you focused.
- **Intuitive Kanban:** Simple drag-and-drop workflow to visualize your progress.

## 🛠 Quick Start

Since it's a web-based tool, you can just click the [Live Demo](https://jinyeong-park.github.io/kanban-board/) link above.

If you want to contribute or run it locally:

```bash
# Clone the repository
git clone [https://github.com/jinyeong-park/kanban-board.git](https://github.com/jinyeong-park/kanban-board.git)

# Navigate to the folder
cd kanban-board

# Open index.html in your browser
```
